# NIBRS-Helper-LLM
A NIBRS AI that can help answer questions about various NIBRS requirements

Step 1:
  Download the NIBRS manual you wish to feed the NIBRS Bot

Step 2:
  Extract the PDF into txt files using PDF Plumber using the extract_pdfs.py script I wrote
    Make sure to set the proper paths before running or it wont work!

Step 3: Run the chunk_text.py script to create chunks for the LLM to eat.
